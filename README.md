# FluentROI

FluentROI is a modern desktop image annotation tool built with **WinUI 3** and **Fluent Design**.  
It allows you to create, move, and resize multiple **ROI (Regions of Interest)** on images, with real-time preview and easy export.  
The core ROI functionality is also available as a **reusable library** for integration into other apps.

**Key Features:**
- Zoom and pan images with mouse
- Draw, select, move, and resize ROI with 8 anchors
- Multi-ROI support
- Real-time ROI preview and export to PNG
- Fluent Design UI (Acrylic/Mica, modern buttons)
- Library / SDK for other applications
